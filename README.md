# Aether3D

Aether3D 是一个基于 **Gaussian Splatting** 的 3D 资产生成与可视化项目，目标是将真实世界的场景快速转换为**可交互、可保存、可分享**的三维空间。

## 自研边界与开源依赖

### 自研边界

Aether3D 的 Tri/Tet 核心几何真值骨架、关键训练与渲染主链路为自研实现。\
边界定义见：`docs/compliance/self_developed_boundary.md`。

### 开源依赖与履约

系统仍使用少量开源基础组件（例如加密与服务框架），并按许可证履约。\
依赖清单见：`compliance/third_party/manifest.v1.json`。\
许可证与声明见：`NOTICE`、`compliance/third_party/licenses/`、`compliance/third_party/notices/`。

### 对外口径

允许与禁止说法见：`docs/compliance/public_claims_policy.md`。
