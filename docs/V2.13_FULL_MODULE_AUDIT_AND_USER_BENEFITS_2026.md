## 二、渲染模块（Rendering）

### 2.1 现状

| 组件 | 状态 | 说明 |
|------|------|------|
| **Core/Rendering** | 空目录 | 仅 .gitkeep，无任何渲染逻辑 |
| **WhiteboxViewerViewController** | Placeholder | 显示 "Rendering not implemented"，无 3D 渲染 |
| **ScanGuidanceRenderPipeline** | ✅ 已实现 | Metal 楔形填充 + 边框，扫描时引导 |
| **GuidanceRenderer** | 热力图已删除 | ScanGuidanceRenderPipeline 楔形填充+边框已覆盖，三角形颜色随 displaySnapshot 变化 |
| **EvidenceRenderer** | 占位 | "Metal Renderer Active" 文案，无实际绘制 |

### 2.2 方法论缺口

- **3D 模型渲染**：需 RealityKit / SceneKit / Metal 渲染 Mesh 或 3DGS splat
- **引导可视化**：ScanGuidanceRenderPipeline 已将 coverage→displaySnapshot→楔形填充实现（热力图已删除）
- **箭头指示**：需根据 PIZ/方向生成箭头叠加

### 2.3 对用户好处（大白话）

- **现在**：拍完的 3D 作品点进去只能看到「渲染未实现」几个字，看不到模型
- **补齐后**：可以旋转、缩放、从各个角度查看自己的 3D 作品

## 五、3D 作品编辑模块

### 5.1 现状

- **完全缺失**：无 mesh 编辑、无变换（缩放/旋转/裁剪）、无材质调整

### 5.2 可自研/自建方案

| 功能 | 方案 | 优先级 |
|------|------|--------|
| 基础变换 | 手势 + 4×4 矩阵（平移/旋转/缩放） | P1 |
| 裁剪/边界框 | AABB 裁剪 + 重导出 | P2 |
| Mesh 简化 | QEM 或边折叠（LOD） | P2 |
| 材质编辑 | 顶点色/贴图替换（若支持） | P3 |

### 5.3 对用户好处（大白话）

- **现在**：作品是什么样就什么样，不能调整
- **补齐后**：可以旋转看细节、裁掉多余部分、简化后导出给 3D 打印等

## 九、结论

- **拍摄链**：模糊检测已升级，其余质量检测仍为 placeholder
- **渲染**：扫描引导有 Metal 实现，作品查看器无渲染
- **UI**：画廊 + 分享可用，引导可视化缺失
- **社交**：系统分享已有，社区/公开展示待建
- **3D 编辑**：全缺，需自研基础变换与裁剪

**用户一句话**：现在能拍、能分享、能导出部分格式，但**还看不到 3D 模型**，也**不能编辑**。补齐 3D 查看器和基础编辑后，体验会完整很多。
