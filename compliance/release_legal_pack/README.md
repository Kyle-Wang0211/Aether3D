# Release Legal Pack

This directory stores release-grade legal evidence for each version.

## Required contents per release

1. `manifest.v1.json` snapshot (copy from `compliance/third_party/manifest.v1.json`).
2. `NOTICE` snapshot.
3. `licenses/` snapshot for all shipped dependencies.
4. `public_claims.txt` approved external wording.
5. `approval_record.md` with legal sign-off references.
